## :sparkles: Overview

This project is a modified version of the original SA-MP Discord Rich Presence plugin.

It introduces small but meaningful improvements, focusing on enhancing the display layout, improving readability, and adding support for custom Indonesian server branding.

This is not a full rewrite — only a refinement and customization of the original work.

---

## :rocket: Features

* Improved Discord Rich Presence display
* Cleaner and more readable layout
* Support for custom Indonesian server logos
* Lightweight and optimized
* Maintains original functionality with enhancements

---

## :jigsaw: Compatibility

* SA-MP 0.3.7 (R1 – R5)
* SA-MP 0.3.DL
* Requires ASI Loader

## :gear: Installation

1. Download the `.asi` file
2. Place it into your GTA San Andreas root directory
3. Make sure ASI Loader is installed
4. Launch SA-MP
5. Enjoy the enhanced Discord Rich Presence
