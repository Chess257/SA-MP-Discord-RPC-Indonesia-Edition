## :sparkles: Overview

This project is an enhanced and modified version of the original SA-MP Discord Rich Presence plugin.

It focuses on improving visual clarity, refining the display layout, and adding support for custom Indonesian server branding to provide a more modern and immersive Discord RPC experience.

---

## :rocket: Features

* Improved Discord Rich Presence display
* Support for Indonesian server logos
* Cleaner and more readable layout
* Lightweight and optimized performance
* Easy integration with SA-MP client

---

## :jigsaw: Compatibility

This plugin is compatible with:

* SA-MP 0.3.7 (R1 – R5)
* SA-MP 0.3.DL
* Standard client setups with ASI Loader

## :gear: Installation

1. Download the `.asi` file
2. Place it into your GTA San Andreas root directory
3. Make sure you have an ASI Loader installed
4. Launch SA-MP
5. Enjoy the enhanced Discord Rich Presence
